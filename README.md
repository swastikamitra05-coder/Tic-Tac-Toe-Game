# Tic Tac Toe AI
A clean and interactive Tic Tac Toe game with a basic AI opponent, built using HTML, CSS, and JavaScript.

## Features
- Play against computer (AI)
- Score tracking system
- Responsive design (mobile-friendly)
- Clean UI

## Tech Stack
- HTML
- CSS
- JavaScript
